# JOIN CHART 구독형 서비스 기획서

## 0. 한 줄 정의
> **"맞춰주는 척하지 않는 차트 분석 도구."**
> 작도법으로 차트 구조를 객관적으로 보여주고, 검증된 통계만 정직하게 제시한다.

---

## 1. 포지셔닝 (가장 중요)

### 팔지 말아야 할 것
- ❌ "주가를 맞춰드립니다" — 검증상 방향 예측은 추세 연장만 맞고 전환점은 못 잡음. 과장하면 신뢰 붕괴.
- ❌ "이 종목 5배 갑니다" 류의 희망

### 팔아야 할 것
- ✅ **객관적 차트 구조** — 작도(지지/저항 교차점), 매물대, 파동 위치, 시나리오 3종
- ✅ **검증된 통계** — "녹색 목표 도달 28%", "위험선 반등 51~57%" 같이 우리가 4.5만 건으로 직접 검증한 숫자만
- ✅ **정직함 자체** — "이건 예측이 아니라 참고"라고 말하는 도구. 리딩방 피로감에 대한 안티테제.

### 슬로건 후보
- "희망 대신 구조를 봅니다"
- "오를지는 모릅니다. 차트가 어떻게 생겼는지는 보여드립니다."
- "리딩방은 답을 팝니다. 우리는 판단 근거를 드립니다."

---

## 2. 타겟 (가격이 타겟을 정한다)

월 3만원+ 가격이면 실질 타겟은 자동으로 좁혀짐:
- **핵심:** 스스로 매매하는 중급 이상. 차트를 진지하게 보고, 리딩방에 데여본 사람.
- **부차:** 종목 많이 보는 사람(여러 종목 빠르게 스캔하고 싶은), 국내+미국 같이 보는 사람.
- **비타겟:** 입문자(3만원 안 냄), "찍어줘" 원하는 사람(우리 철학과 안 맞음 — 오면 실망하고 나감).

> 입문자까지 norm먹으려면 **무료 체험 + 저가 입문 플랜**으로 깔때기를 만들되, 메인 수익은 중급 이상에서.

---

## 3. 기능 → 플랜 분배

### 무료 (체험·유입)
- 하루 3~5종목 검색 제한
- 일봉 차트 + 작도 + 기본 시나리오
- "이 도구가 뭘 하는지" 맛보기

### 베이직 (예: 월 9,900~14,900원)
- 검색 무제한 (국내)
- 일봉 + 10분봉 두 트랙
- 작도·시나리오·매물대·위험선 전부
- 4/1 분기 기준 시나리오 비교

### 프리미엄 (월 3만원+)
- 베이직 전부 +
- **미국 주식** (야후 연동)
- **검증 통계 리포트** — 녹색 도달률, 위험선 반등률 등 우리만의 백테스트 수치
- **관심종목 대시보드** — 여러 종목 작도 상태 한눈에 (오를 자리/위험 자리 스캔)
- **분기 리셋 알림** — 7/1 같은 기준 전환 시점 알림
- (향후) 종목별 시나리오 도달 추적 — "4/1에 그린 시나리오, 지금 어디로 갔나"

---

## 4. 가격 정당화 (월 3만원을 어떻게?)

정직함을 파는데 비싸게 받으려면 **"근거의 질"**로 정당화해야 함:

1. **검증된 통계** — 남들은 감으로 "지지선이다" 하는데, 우리는 "위험선 반등 51~57%(2만건 검증)"처럼 숫자로. 이게 3만원의 핵심 명분.
2. **국내+미국 통합** — 두 시장 한 곳에서. 다른 도구 두 개 구독할 거 하나로.
3. **시간 절약** — 관심종목 대시보드로 여러 종목 작도 상태를 빠르게 스캔. 트레이더의 시간 = 돈.
4. **광고·리딩 없음** — 중립적. 누가 돈 줬다고 종목 밀지 않음.

> 핵심 메시지: "월 3만원으로, 매매 한 번 실수 안 하면 본전 뽑고 남는다."

## 5. 리스크 & 대응

| 리스크 | 대응 |
|---|---|
| "정직한데 왜 돈 내?" | 정직함=무료가 아니라, **검증 데이터와 도구의 질**이 유료. 정직은 신뢰의 기반일 뿐. |
| 안 맞으면 환불·욕 | 애초에 "예측 아님" 명시. 맞춤 보장 안 함을 약관·UI에 박기. 기대 관리가 곧 방어. |
| LS/야후 API 의존 | 데이터 끊기면 서비스 정지 리스크. 다중 소스 백업 검토. |
| 투자자문업 규제 | **"매수/매도 추천" 절대 금지.** 차트 정보 제공 도구로 선 긋기. (법률 검토 필수) |
| 경쟁(증권사 HTS 무료 차트) | 차별점 = 작도법 + 검증통계 + 정직 포지션. HTS엔 없는 것. |

### ⚠️ 가장 큰 법적 리스크
한국에서 **유료로 투자 정보·신호를 제공하면 유사투자자문업 신고** 대상일 수 있음. "차트 분석 도구"라도 구독료를 받으면 걸릴 수 있으니 **반드시 법률·금감원 확인**. 이거 안 하면 서비스 자체가 위험.

## 6. 출시 로드맵 (제안)

**1단계 — MVP (현재 거의 완성)**
- 국내 작도 차트 + 시나리오 + 검증통계. 무료로 풀고 사용자 반응·피드백 수집.

**2단계 — 결제·플랜 붙이기**
- 로그인/회원 + 결제(토스페이먼츠 등) + 플랜별 기능 제한. 미국 주식 프리미엄 분리.

**3단계 — 차별화 기능**
- 관심종목 대시보드, 분기 리셋 알림, 시나리오 도달 추적.

**4단계 — 신뢰 콘텐츠**
- "이번 분기 작도 적중 리포트" 정기 공개 → 정직함을 마케팅으로. (맞은 것도 틀린 것도 공개 = 신뢰)
